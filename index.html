Pengaturan Email (EmailJS)

Cara Setup (Gratis):
1. Daftar di emailjs.com (gratis)
2. Email Services → Add Service (Gmail/Outlook) → salin Service ID
3. Email Templates → Create Template:
   • Field "To Email" → isi: {{to_email}}
   • Field "Subject" → isi: {{subject}}
   • Field "Content" → isi: {{{message}}} (3 kurung kurawal untuk HTML)
   • Simpan → salin Template ID
4. Account → General → salin Public Key
